## わかったこと

- 便利コマンド 30 種類
- シェルとは？
  - ターミナルからカーネルにコマンドを送るときに中継地点としてシェルがある。**zsh や bash などこれらを Shell という。**
- Neovim と Vim とちらがいいか？
  - 結論 Neovim。使い方もほぼ一緒で vim より拡張性もいいから上位互換。
- vim の便利プラグイン
  - NERDTree
    - ディレクトリのツリー構成が見れる。可視化しやすく便利。
  - FyzzyFinder
    - ノーマルモードにて`:FZF`とコマンドを打つことでお探しのファイルが見つけられる
  - vim-fugitive
    - git コマンドが vim 上で打てるプラグイン
    - `git blame ファイル名`このコマンドはいつ誰がどの行をコミットしたか見えるコマンドチーム開発ではよく使われるコマンド
  - vim-gitgutter
    - GitHub で管理してるファイルのコミットした後と前の差分を Neovim で可視化することができる。
      これもまたチーム開発で生きるプラグインだと思います。
  - vim-comentary
    - これは、選択している部分を一気にコメントアウトできる。
      コマンド＋スラッシュで同じことできるがなれたらこちらの方がいいのですかね？
